Detailed Description

template<typename V = void, typename E = void>
template<Constness Const>
struct graph::DirectedGraph< V, E >::IncidentEdgeRange< Const >

A range capturing all the edges incident to a vertex in a given direction.

The range is cyclical and will eventually return to the starting edge.

Template Parameters
ConstWhether the referenced edges are const or mutable.
