Detailed Description

template<typename V, typename E>
struct graph::DirectedGraph< V, E >::EdgeNode

A node in a linked list of edges.

Each edge node links to the next edge in two lists: One of the outgoing edges around the source vertex, and one of the incoming edges around the target vertex.

If the edge is the last in the list, it points to itself.

If this graph stores edge values, the edge data is stored in data.
