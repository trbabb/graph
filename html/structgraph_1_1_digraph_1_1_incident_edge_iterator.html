Detailed Description

template<typename V = void, typename E = void, template< class... > class Map = std::unordered_map>
template<Constness Const>
struct graph::Digraph< V, E, Map >::IncidentEdgeIterator< Const >

An iterator over the edges incident to a vertex.

IncidentEdgeIterators can be compared for equality with EdgeIterators; the direction of traversal is ignored; the iterators are equal iff they point to the same edge.

Template Parameters
ConstWhether the referenced edges are mutable or const.
