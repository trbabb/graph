Detailed Description

template<typename V, typename E>
struct graph::DirectedGraph< V, E >::VertexNode

A node holding information about a vertex.

Each vertex node carries the head and tail of two linked lists of edges: One of the incoming edges around the vertex, and one of the outgoing edges.

If there are no edges in a list, the head and tail point to std::nullopt.

If this graph stores vertex values, the vertex data is stored in data.
